# 🍕 Pizza Sales Analysis

## 📌 Project Overview

Pizza Sales Analysis is a data analytics project that examines pizza sales data to uncover valuable business insights. Using SQL, the project analyzes sales performance, customer purchasing patterns, revenue trends, and product popularity to support data-driven decision-making.

## 🚀 Features

* Sales performance analysis
* Revenue and order trend evaluation
* Best-selling and least-selling pizza identification
* Category-wise sales analysis
* Size-wise sales distribution
* Daily and monthly sales trends
* KPI dashboard creation

## 🛠️ Technologies Used

* SQL
* MySQL
* Data Analysis
* Business Intelligence
* Data Visualization

## 🎯 Project Objectives

* Analyze overall sales performance
* Identify top-performing pizza categories
* Track revenue generation trends
* Understand customer ordering behavior
* Discover business growth opportunities
