# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview

Pizza Sales Analysis Dashboard is an end-to-end Data Analytics project that analyzes pizza sales data to generate actionable business insights. The project utilizes Excel for data cleaning and validation, SQL/MySQL for data analysis, and Power BI for interactive dashboard development and visualization.

The dashboard helps stakeholders understand sales performance, customer ordering behavior, revenue trends, product popularity, and business growth opportunities through interactive reports and KPIs.

---

## 🚀 Project Workflow

### 1. Data Collection

* Imported pizza sales dataset from Excel files.
* Verified data consistency and completeness.

### 2. Data Cleaning & Preparation

* Cleaned and validated data using Microsoft Excel.
* Handled missing values and data inconsistencies.
* Standardized date and category fields.

### 3. Data Analysis

* Performed SQL queries in MySQL to analyze sales performance.
* Used aggregate functions, joins, grouping, and filtering techniques.
* Extracted business insights from raw transactional data.

### 4. Dashboard Development

* Built an interactive Power BI dashboard.
* Created KPIs, charts, slicers, and business reports.
* Implemented dynamic filtering and drill-down analysis.
